Berdasarkan penugasan, telah dilakukan tinjauan terhadap 16 jurnal terindeks Scopus/WoS Q1-Q3 yang berkaitan dengan akuisisi data, penjadwalan real-time, arsitektur memori hibrid, dan analisis energi pada mikrokontroler ARM Cortex-M.

Ringkasan komprehensif (Future Work) dari ke-16 jurnal tersebut kemudian digabungkan untuk menghasilkan suatu gagasan metode baru yang komprehensif. Daftar lengkap jurnal dan \textit{future work} terlampir pada file repository proyek.

\section{Fase 2: Perancangan Metode Baru}
Berdasarkan sintesis \textit{future work}, diusulkan metode baru dengan judul:
\begin{quote}
\textbf{"Adaptive Safety-Critical Multi-Sensor Acquisition Framework with Priority-Deadline Hybrid Scheduling and Hybrid Memory Optimization on STM32F407VGTx"}
\end{quote}

\subsection{Komponen Utama Sistem}
Sistem ini mengintegrasikan beberapa pendekatan:
\begin{enumerate}
    \item \textbf{Adaptive ADC DMA:} Kecepatan sampling bervariasi bergantung pada dinamika sinyal.
    \item \textbf{Hybrid Memory Placement:} Pemanfaatan CCM (Core Coupled Memory) untuk buffer kritis.
    \item \textbf{Priority-Deadline Scheduler:} Penjadwalan FreeRTOS dengan pemisahan tugas berdasarkan urgensi dan batas waktu.
    \item \textbf{Safety Barrier Monitoring:} Penggunaan WWDG dan Control Barrier Function (CBF) untuk mitigasi kegagalan.
\end{enumerate}

\section{Fase 3: Simulasi dan Implementasi}
Sistem diimplementasikan menggunakan STM32CubeMX dan Keil $\mu$Vision. Karena keterbatasan model pada simulator, pengujian dilakukan menggunakan mikrokontroler \textbf{STM32F401VE} di Proteus yang memiliki arsitektur Cortex-M4 serupa.

\subsection{Konfigurasi Sistem RTOS}
Sistem menjalankan empat \textit{task} utama secara paralel:
\begin{itemize}
    \item \texttt{Task\_SafetyCheck} (Prioritas: \textit{Realtime}) - Memonitor ambang batas tegangan.
    \item \texttt{Task\_SensorRead} (Prioritas: \textit{High}) - Mengambil data ADC dengan kecepatan adaptif.
    \item \texttt{Task\_DataProcess} (Prioritas: \textit{Normal}) - Pemfilteran data dan kontrol PWM aktuator.
    \item \texttt{Task\_CommOutput} (Prioritas: \textit{Low}) - Komunikasi UART dan I2C.
\end{itemize}

\subsection{Pengujian Perangkat Keras}
Pada simulasi Proteus, dua potensiometer bertindak sebagai representasi dua sensor analog. Mikrokontroler berhasil mengeksekusi konversi ADC via DMA, memfilter data dengan rata-rata bergerak (\textit{moving average}), dan mengatur siklus kerja (\textit{duty cycle}) pada pin keluaran PWM dengan frekuensi 10 kHz. 

Fitur \textit{watchdog timer} (WWDG) dan CBF berfungsi optimal, terbukti saat simulasi diarahkan ke nilai batas (0V atau 3.3V), sistem masuk ke fase \textit{FAULT} dan mengamankan \textit{output}.
